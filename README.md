# LyricsXForFlexbar

LyricsXForFlexbar is a [LyricsX](https://github.com/MxIris-LyricsX-Project/LyricsX) fork maintained by [perryliu27](https://github.com/perryliu27). It keeps the original macOS lyrics experience and adds Flexbar integration for showing now playing information and lyrics on Flexbar.

The upstream project is maintained by [Mx-Iris](https://github.com/Mx-Iris), and the original LyricsX project was created by [Xander Deng / ddddxxx](https://github.com/ddddxxx/LyricsX).

The companion Flexbar plugin is `Flexbar-LyricsXForFlexbarPlugin`.

## Features

- Adds a local HTTP service that exposes now playing and lyrics data for Flexbar.

## Requirements

- macOS 11+
